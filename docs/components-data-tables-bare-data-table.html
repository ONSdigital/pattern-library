Developer Notes

All data-tables are based on Filament Group's Tablesaw library, which provides an out of the box solution to responsive tables.

Out of the box, the library features 3 different table types; Swipe table, Toggle table and Bare table. It also has a mode which allows switching between the different types.

The extent to the customisations for use within the pattern library is limited - specifically, the only additions are to present the tables in keeping with the of the pattern libraries look.

Although no swipe or toggle controls are used in this pattern, it is still required that the class="tablesaw" is used to ensure it works responsively.
